## Map Loader
### ```google-map-loader```
This component is a ready to use configurable map that supports the use of map markers, centering and other features. It automatically uses the [api loader](../api-loader/about.story.md) to load the google maps api. Then it uses the [map container](../map-container/about.story.md) to load a map instance on an element.
