## Popper
#### ```pockets-popper```

The popper component is a user interface element that shows a small, floating container with content when a user interacts with a target element. It can be used to display tooltips, dropdowns, or other contextual information without cluttering the main interface. The popper appears adjacent to the target element and the positioning can be controlled.
