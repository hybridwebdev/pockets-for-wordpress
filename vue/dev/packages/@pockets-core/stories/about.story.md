## About
 
Pockets is a framework for organizing application logic and data into structured, reusable components. It provides conventions and utilities for defining models, handling data operations, and structuring code in a consistent way.

The goal of Pockets is to simplify how data-driven features are built and maintained by introducing clear patterns for interacting with models, queries, and related logic. This approach helps keep application code modular, predictable, and easier to navigate as projects grow.
