## Pockets Nodes - Tree Router View 

This component is a simple wrapper that loads and renders data for the current route. 
When a route changes, it loads the coresponding data for that route. It will render the new document tree and also update the document title. 

It also does basic pre-fetching when a user hovers over a link. This only applies to URLS for the current site.
